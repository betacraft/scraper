# Scraper for parsing app information from Google Play Store

## How to use?
```go
import "github.com/betacraft/scraper"

func xyz(){
   app, err := scraper.Scrape("https://play.google.com/store/apps/details?id=com.yourpackage.name")
   if err != nil {
       //parsing failed
   }
}
```
## App struct
```go
type App struct {
	Name        string

```
